# Ghostnet
Reticulum Based Messenger inspired by S2Underground's work to bring communities together during uncertain times over infrastructure-free mediums.

This project is an Ubuntu/Debian based instance that serves a UI for managing RNS direct messaging, group messaging, propogation nodes and RNS over VHF/UHF/HF and Rnode (LoRa) radio.

This was tested using the DigiRig Lite, DigiRig Mobile, Yaesu SCU-17 radio modems as well as various radios and TCP network configurations.

Releases contain the deb and tarball for online and offline installation.
